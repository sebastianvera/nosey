### Support

#### Github

Language   | Supported | Notes
---        | ---       | ---
Javascript | ✔         | `require` and `import` are supported.
Vim        | ✔         | `vim-plug` and `vundle` are supported.
Go         | ✔         | `single`, `grouped`, `named`, `.` and `_` imports are supported.
Ruby       | ✖️         |
Elixir     | ✖️         |
